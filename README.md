## StudyGen - An AI that turns PDF into likely Exam questions. 
The Core idea is the that it takes any pdf such as lecture notes, texts book, reasearch papers etc and turns it into smart study partner generating likely exam questions, quizzes, summaries and personalized practice powered by AI.

## Tech Stack
- Python
- Django 

## Setup Instructions
1. Clone the repository
2. Create and activate a virtual environment
- python -m venv venv
- venv\Scripts\activate
3. Install dependencies
4. Run migrations 
5. Start the development server
- python manage.py runserver
